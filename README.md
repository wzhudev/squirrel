A no (or low) config bundler for bundling Typescript libraries.

## Usage

Install

```sh
yarn add @wendellhu/squirrel -D
```

Build

```sh
npx run squirrel
```

Or adding this to your package.json file

```diff
"scripts": {
+  "build": "squirrel"
"
```

and run

```sh
yarn build
```
